# 常用的步骤

## 部署网站

> 前提：
> 1. 编写好的前端代码（这里用react）
> 2. 编写好的服务端代码 （这里用eggjs）
> 3. 本地数据库（这里用mongodb）
> 4. 服务器
> 5. 备案域名，或者国外不备案
> 


### 1. 启动服务器  ssh root@ip

* 这里需要设置一下 SELinux，以及打开防火墙

```shell
# SELinux 部分
# 打开 SELinux 有三种模式：Disabled、Permissive 和 Enforcing

# 查看当前 SELinux 状态
getenforce

# 设置 SELinux，设置完成后重新启动就可以了
vim /etc/selinux/config

# This file controls the state of SELinux on the system.
# SELINUX= can take one of these three values:
#     enforcing - SELinux security policy is enforced.
#     permissive - SELinux prints warnings instead of enforcing.
#     disabled - No SELinux policy is loaded.
SELINUX=permissive # 这里设置模式
# SELINUXTYPE= can take one of three values:
#     targeted - Targeted processes are protected,
#     minimum - Modification of targeted policy. Only selected processes are protected.
#     mls - Multi Level Security protection.
SELINUXTYPE=targeted


# 防火墙部分
# 查看防火墙，如果是 dead 状态，即防火墙未开启
systemctl status firewalld
# 打开防火墙
systemctl start firewalld
# 关闭防火墙
systemctl stop firewalld
# 开放端口
firewall-cmd --permanent --zone=public --add-port=7001/tcp
```

### 2. 安装数据库（mongodb）

```shell
# 第一步：创建.repo文件，生成mongodb的源
vi /etc/yum.repos.d/mongodb-org-4.2.repo

# 第二步：添加配置信息
[mongodb-org-4.4]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/4.4/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-4.4.asc
  # name：指的是名称
  # baseurl：获得下载的路径
  # gpgcheck：表示对从这个源下载的rpm包进行校验
  # enabled：表示启用这个源
  # gpgkey：gpg验证

# 第三步：使用yum安装Mongodb
sudo yum install -y mongodb-org

# 第四步：获取安装包
rpm -qa | grep mongodb

# 第五步：获取安装路径
rpm -ql mongodb-org-server

# 第六步：启动服务
systemctl start mongod.service

# 第七步：查看端口 mongodb 默认27017
netstat -natp | grep 27017

# 第八步：查看数据库的进程是否存在
ps -aux | grep mongod

# 第九步：验证服务
mongo

# 第十步：关闭服务
use admin;
db.shutdown(); # 这样执行后需要重新开启 mongodb 服务


# 其他常用命令总结：命令使用同一类型的
# 开启服务
systemctl start mongod.service
sudo service mongod start
# 关闭服务
sudo service mongod stop
# 重启服务
sudo service mongod restart
# 开机启动
sudo chkconfig mongod on
# 卸载
sudo yum erase $(rpm -qa | grep mongodb-org)
# 开放端口，这个需要先打开防火墙
iptables -A INPUT -p tcp -m state --state NEW -m tcp --dport 27017 -j ACCEPT
# 端口开放后记得在云服务商的防火墙把对应端口打开


# 本地跑mongodb
mongod --config /usr/local/mongodb/etc/mongodb.conf
mongo
show dbs # 查看当前数据库
use active # 切换到指定数据库，如果没有则创建
db # 显示当前使用的数据库
db.active.find() # 查看当前数据
show collections # 查看集合

# 退出
use admin
db.shutdownServer()



brew services start mongodb-community@5.0
brew services stop mongodb-community@5.0
```

### 3. 安装 Node，Nginx

```shell
# Nginx
# 第一步：安装yum-utils
sudo yum install yum-utils
# 第二步：配置信息
vim /etc/yum.repos.d/nginx.repo

[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key
module_hotfixes=true

[nginx-mainline]
name=nginx mainline repo
baseurl=http://nginx.org/packages/mainline/centos/$releasever/$basearch/
gpgcheck=1
enabled=0
gpgkey=https://nginx.org/keys/nginx_signing.key
module_hotfixes=true

# 第三步：安装nginx
sudo yum install nginx

# 其他常用命令总结：命令使用同一类型的
# 测试 nginx 配置文件
nginx -t
# 停止、开启、重启
systemctl stop nginx.service
systemctl start nginx.service
systemctl restart nginx.service
# 查看当前状态
systemctl status nginx.service
# 开机自启
systemctl enable nginx.service
# 取消开机自启
systemctl disable nginx.service


# Node
curl -fsSL https://rpm.nodesource.com/setup_16.x | sudo bash -
sudo yum install nodejs
node -v
npm -v

```
### 4. 打包上传 前后端上传服务器

```shell
# 服务端打包 这里已 eggjs 为参考
# 打包
tar -zcvf ../release.tgz .
# 解压
tar zxvf ../release.tgz
 
# 前端打包 这里已 react 为参考
npm run build

# 上传服务器
# scp 上传文件 root@ip地址:服务器地址
scp release.tgz root@ip:/data/active/...

```

#### 发布前后端

```shell
# 发布服务端 这里已 eggjs 为参考
# 安装线上依赖，可以在本地也可以在服务器上
npm i --production
npm run tsc
npm start # 项目跑起来后可以在内网 curl 试试通不通

  # eggjs config添加
  config.cluster = {
    listen: {
      path: '',
      port: 7001,
      hostname: '0.0.0.0',
    },
  };

npm stop

# 发布前端，理论上前端算是静态资源直接 nginx 反向代理就行

# Nginx配置 这里算是一个服务器部署前后端，正常来说前后端需要部署不同服务器，用不同的域名代理
server {
    listen       80; # 监听 80 端口
    server_name www.fbyron.cn;
    root /data/active/active-life/; # 当输入域名后反向代理到active-life/index.html
    index index.html index.htm;
    location ~* ^/byron { # 当监听到域名/byron 后反向代理到主机的 127.0.0.1:7001
      proxy_pass http://127.0.0.1:7001; # 代理本地地址
      proxy_connect_timeout 15s;
      proxy_send_timeout 15s;
      proxy_read_timeout 15s;
      proxy_set_header X-Real-IP $remote_addr;
      proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
      rewrite "^/byron/(.*)$" /$1 break; # 将请求的 /byron 移除掉
      #root /data/active/active-life/;
      #index index.html index.htm;
    }
}
# 配置 config 后 reload 开始测试

```
